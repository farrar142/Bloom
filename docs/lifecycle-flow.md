### 코드 예시

```python
from bloom import Application, Component
from bloom.core.decorators import PostConstruct, PreDestroy

@Component
class DatabaseService:
    @PostConstruct
    async def connect(self):
        """비동기 초기화 - start_async()에서 실행"""
        await self.pool.connect()
        print("DB Connected")
    
    @PreDestroy
    async def disconnect(self):
        """비동기 정리 - shutdown_async()에서 실행"""
        await self.pool.close()
        print("DB Disconnected")

@Component
class CacheService:
    @PostConstruct
    def init_cache(self):
        """동기 초기화 - ready()에서 실행"""
        self.cache = {}
        print("Cache Initialized")
    
    @PreDestroy
    def clear_cache(self):
        """동기 정리 - shutdown_async()에서 실행"""
        self.cache.clear()
        print("Cache Cleared")

# 애플리케이션 생성 및 실행
app = Application("myapp")
app.scan(DatabaseService, CacheService)
app.ready()  # 동기 @PostConstruct 실행

# ASGI 앱으로 uvicorn에서 실행
# uvicorn main:app.asgi
# → lifespan.startup에서 start_async() 자동 호출
# → lifespan.shutdown에서 shutdown_async() 자동 호출
```

### 미들웨어 상세 흐름

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                       MIDDLEWARE PROCESSING DETAIL                           │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  Request ──►                                                                │
│                                                                             │
│  ┌─────────────────────────────────────────────────────────────────────┐    │
│  │  MiddlewareProcessContext.__aenter__()                              │    │
│  │                                                                     │    │
│  │  for middleware in chain:  # 순서대로                               │    │
│  │      │                                                              │    │
│  │      ├─────────────────────────────────────────────────────────┐    │    │
│  │      │  Generator Middleware (yield 기반)                       │    │    │
│  │      │                                                          │    │    │
│  │      │  async def __call__(self, request, ctx):                │    │    │
│  │      │      # 요청 단계 (yield 전)                              │    │    │
│  │      │      request.state.start_time = time.time()             │    │    │
│  │      │                                                          │    │    │
│  │      │      response = yield  ◄── 여기서 일시 정지              │    │    │
│  │      │                                                          │    │    │
│  │      │      # 응답 단계 (yield 후) - 나중에 실행                 │    │    │
│  │      │      duration = time.time() - request.state.start_time  │    │    │
│  │      │      response.headers["X-Duration"] = str(duration)     │    │    │
│  │      │      return response                                     │    │    │
│  │      │                                                          │    │    │
│  │      └─────────────────────────────────────────────────────────┘    │    │
│  │      │                                                              │    │
│  │      ├─────────────────────────────────────────────────────────┐    │    │
│  │      │  Early Return Middleware (인증 실패 등)                   │    │    │
│  │      │                                                          │    │    │
│  │      │  async def __call__(self, request, ctx):                │    │    │
│  │      │      if not is_authenticated(request):                  │    │    │
│  │      │          ctx.early_response = JsonResponse(             │    │    │
│  │      │              {"error": "Unauthorized"}, status=401      │    │    │
│  │      │          )                                               │    │    │
│  │      │          return  # 이후 미들웨어/핸들러 스킵             │    │    │
│  │      │      yield                                               │    │    │
│  │      │                                                          │    │    │
│  │      └─────────────────────────────────────────────────────────┘    │    │
│  │      │                                                              │    │
│  │      └──► ... 다음 미들웨어들                                       │    │
│  │                                                                     │    │
│  └─────────────────────────────────────────────────────────────────────┘    │
│                             │                                               │
│                             ▼                                               │
│                      [Handler 실행]                                         │
│                             │                                               │
│                             ▼                                               │
│  ┌─────────────────────────────────────────────────────────────────────┐    │
│  │  MiddlewareProcessContext.__aexit__()                               │    │
│  │                                                                     │    │
│  │  for middleware in reversed(chain):  # 역순으로                     │    │
│  │      │                                                              │    │
│  │      ├─ generator.asend(response)  # yield 이후 코드 실행          │    │
│  │      │                                                              │    │
│  │      └─ response = 수정된 응답                                      │    │
│  │                                                                     │    │
│  └─────────────────────────────────────────────────────────────────────┘    │
│                             │                                               │
│                             ▼                                               │
│                                                                 ──► Response│
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### CALL_SCOPED 상세 흐름

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                    CALL_SCOPED DETAILED FLOW                                 │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  @Controller                                                                │
│  class UserController:                                                      │
│      service: UserService          # DI 주입                                │
│                                                                             │
│      @Get("/users/{id}")                                                    │
│      async def get_user(self, id: int):                                     │
│          return await self.service.get(id)                                  │
│                                                                             │
│  @Component                                                                 │
│  class UserService:                                                         │
│      repo: UserRepository          # DI 주입                                │
│      session: Session              # CALL_SCOPED                            │
│                                                                             │
│  @Component                                                                 │
│  class UserRepository:                                                      │
│      session: Session              # CALL_SCOPED (같은 세션!)               │
│                                                                             │
│  ════════════════════════════════════════════════════════════════════════   │
│                                                                             │
│  HTTP GET /users/1                                                          │
│         │                                                                   │
│         ▼                                                                   │
│  ┌──────────────────────────────────────────────────────────────────────┐   │
│  │  async with call_scope(controller, "get_user"):                      │   │
│  │       │                                                              │   │
│  │       │  ┌─────────────────────────────────────────────────────┐    │   │
│  │       ├─►│ 1. UserService 주입                                  │    │   │
│  │       │  │    - session 필요                                    │    │   │
│  │       │  │    - get_scoped_prototype(Session) → None            │    │   │
│  │       │  │    - Session() 생성 (id=1)                           │    │   │
│  │       │  │    - set_scoped_prototype(Session, session)          │    │   │
│  │       │  │    - session.__enter__() 호출 ✓                      │    │   │
│  │       │  └─────────────────────────────────────────────────────┘    │   │
│  │       │                                                              │   │
│  │       │  ┌─────────────────────────────────────────────────────┐    │   │
│  │       ├─►│ 2. UserRepository 주입                               │    │   │
│  │       │  │    - session 필요                                    │    │   │
│  │       │  │    - get_scoped_prototype(Session) → session(id=1)   │    │   │
│  │       │  │    - 캐시된 인스턴스 반환 ✓                          │    │   │
│  │       │  └─────────────────────────────────────────────────────┘    │   │
│  │       │                                                              │   │
│  │       │  ┌─────────────────────────────────────────────────────┐    │   │
│  │       ├─►│ 3. Handler 실행                                      │    │   │
│  │       │  │    service.get(1)                                    │    │   │
│  │       │  │      └─► repo.find(1)                                │    │   │
│  │       │  │            └─► session.execute(...)                  │    │   │
│  │       │  │                  ↑                                   │    │   │
│  │       │  │                  같은 session 인스턴스!               │    │   │
│  │       │  └─────────────────────────────────────────────────────┘    │   │
│  │       │                                                              │   │
│  │       ▼                                                              │   │
│  │  call_scope 종료 (pop_frame)                                         │   │
│  │       │                                                              │   │
│  │       │  ┌─────────────────────────────────────────────────────┐    │   │
│  │       └─►│ 4. 캐시 정리                                         │    │   │
│  │          │    - _cleanup_scoped_cache_at_depth(0)              │    │   │
│  │          │    - session.__exit__(None, None, None) 호출 ✓      │    │   │
│  │          │    - 트랜잭션 커밋/롤백                              │    │   │
│  │          └─────────────────────────────────────────────────────┘    │   │
│  └──────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```
